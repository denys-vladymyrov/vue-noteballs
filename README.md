The following technologies have been used:
1. Vue Js 3.
2. vue-router
3. pinia (state management)
5. firebase
6. bulma

Live example: https://noteballs.denys-vladymyrov.com/

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
